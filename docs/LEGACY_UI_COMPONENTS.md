# Legacy UI Components (Do Not Use for New Work)

The components below are retained only to keep pre-refactor pages functional during incremental migration.

## Rule
- These files are `LEGACY` and should not be used for any new UI implementation.
- New UI must use the tokenized design-system primitives in `components/ui/*` (foundation set).
